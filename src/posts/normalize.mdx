常使用現成的 `Normalize.css` 標準化樣式，探究一下此套件重設了哪些樣式。

原檔案 [Normalize.css](https://github.com/necolas/normalize.css/blob/master/normalize.css)

## Document

```css
html {
  line-height: 1.15; /* 1 */
  -webkit-text-size-adjust: 100%; /* 2 */
}
```
1. `line-height` 為解決不同瀏覽器預設不同的問題。
https://developer.mozilla.org/zh-CN/docs/Web/CSS/line-height
![](https://i.imgur.com/tznKgO4.png)

2. `text-size-adjust` 防止IOS上方向轉換之後調整字體大小。

```css
  line-height: 1.15; /* 1 */
```

1. 改變所有瀏覽器字體樣式
2. 針對 Safari, Firebox 移除間距

```css
}
```
* 於 IE 顯示 overflow
1. 於 Edge 顯示 overflow

---

```css

  text-transform: none;
}
```
* 移除 Edge, Firefox, IE `text-transform`
1. 移除 Firefox `text-transform`

---

```css
button,
[type="button"],
[type="reset"],
[type="submit"] {
  -webkit-appearance: button;
}
```
* 修 IOS正, Safari 無法設定可點擊樣式

---

```css
button::-moz-focus-inner,
[type="button"]::-moz-focus-inner,
[type="reset"]::-moz-focus-inner,
[type="submit"]::-moz-focus-inner {
  border-style: none;
  padding: 0;
}
```
* 移除 Firefix 中 `inner border`, `padding`

---

```css
button:-moz-focusring,
[type="button"]:-moz-focusring,
[type="reset"]:-moz-focusring,
[type="submit"]:-moz-focusring {
  outline: 1px dotted ButtonText;
}
```
* 恢復 `focus` 未設置樣式

--- 

 ```css

  padding: 0.35em 0.75em 0.625em;
}
 ```
 * 針對 Firefox 一致設定 `padding`

---

```css
legend {
  box-sizing: border-box; /* 1 */
  color: inherit; /* 2 */
  display: table; /* 1 */
  max-width: 100%; /* 1 */
  padding: 0; /* 3 */
  white-space: normal; /* 1 */
}
```
1. 針對 IE, Edge 設定正確的`text wrapping`
2. 針對 IE 設定正確的繼承
3. 刪除所有 `fieldset` 的 `padding`，以便無內容時不會被發現

---

```css
progress {
  vertical-align: baseline;
}
```
* 針對 Chrome, Firefox, Opera 設定正確的垂直置中

---

```css
textarea {
  overflow: auto;
}
```
* 針對 IE 10+ 移除預設的垂直 `Scrollbar`

---

```css
[type="checkbox"],
[type="radio"] {
  box-sizing: border-box; /* 1 */
  padding: 0; /* 2 */
}
```
1. 針對 IE10 設定正確的 `box-sizing`
2. 針對 IE10 移除 `padding`

---

```css
[type="number"]::-webkit-inner-spin-button,
[type="number"]::-webkit-outer-spin-button {
  height: auto;
}
```
* 針對 Chrome 設定正確增量和減量的 `cursor樣式`

---

```css
[type="search"] {
  -webkit-appearance: textfield; /* 1 */
  outline-offset: -2px; /* 2 */
}
```
1. 針對 Chrome, Safari 更正奇怪的外觀
2. 針對 Safari 修正 `outline` 樣式

---

```css
[type="search"]::-webkit-search-decoration {
  -webkit-appearance: none;
}
```
* 針對 Chrome, Safari on Macos，移除內部 `padding`。

---

```css
::-webkit-file-upload-button {
  -webkit-appearance: button; /* 1 */
  font: inherit; /* 2 */
}
```
1. 針對 IOS, Safari 修正無法設定可點擊類型樣式的問題
2. 針對 Safari 設定字型為繼承
