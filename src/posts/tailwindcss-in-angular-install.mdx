簡單紀錄安裝 Tailiwingcss 於 Angular 中的步驟。

1. 新建 Angular APP
```bash
ng new my-app
```

2. 安裝需要的 library
```bash
npm install -D tailwindcss@latest postcss@latest autoprefixer@latest
```

3. 初始化 Tailiwingcss config
```bash
npx tailwindcss init -p
```

4. 新建 postcss.config.js
```javascript
module.exports = {
  plugins: {
    tailwindcss: {},
    autoprefixer: {},
  },
}
```

5. 引入 Tailwind 於 CSS 檔案
```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

5. Done!
